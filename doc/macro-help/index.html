Macro help

Héctor Galbis Sanchis <hectometrocuadrado@gmail.com>

 (require macro-help) package: macro-help

Esta librería ofrece numerosas funciones auxiliares para el manejo de objetos sintácticos. Cada una de las funciones se asegura de mantener el contexto en el que se crearon cada uno de los objetos.

1 Tipos de objetos sintácticos

Las siguientes funciones permiten al programador distinguir diferentes tipos de objetos sintácticos.

procedure

(stx-identifier? v) → boolean?

  v : any/c
Devuelve #t si v es un objeto sintáctico y (syntax-e v) produce un símbolo.

Examples:
> (stx-identifier? #'hola)

#t

> (stx-identifier? #'(casa hola))

#f

> (stx-identifier? #'3)

#f

procedure

(stx-literal? stx) → boolean?

  stx : syntax?
Devuelve #t si (syntax->datum stx) verifica que no es null? y no es pair?.

Examples:
> (stx-literal? #'hola)

#t

> (stx-literal? #'(casa hola))

#f

> (stx-literal? #'3)

#t

procedure

(stx-keyword? stx) → boolean?

  stx : syntax?
Devuelve #t si stx contiene un keyword.

Examples:
> (stx-keyword? #'#:hola)

#t

> (stx-keyword? #'(hola))

#f

> (stx-keyword? #'hola)

#f

2 Constructores y selectores de pares

procedure

(stx-pair? stx) → boolean?

  stx : syntax?
Devuelve #t si stx contiene un datum que es pair?.

Examples:
> (stx-pair? #'(a . b))

#t

> (stx-pair? #'(a b c))

#t

> (stx-pair? #'hola)

#f

procedure

(stx-null? stx) → boolean?

  stx : syntax?
Devuelve #t si stx contiene un datum que es null?.

Examples:
> (stx-null? #'())

#t

> (stx-null? #'(a b c))

#f

> (stx-null? #'hola)

#f

procedure

(stx-cons stx1 stx2) → stx-pair?

  stx1 : syntax?
  stx2 : syntax?
Devuelve el objeto sintáctico formado por un par con los objetos stx1 y stx2.

Examples:
> (stx-cons #'a #'b)

#<syntax:main.rkt:38:5 (a . b)>

> (stx-cons #'hola #'())

#<syntax:main.rkt:38:5 (hola)>

> (stx-cons #'a #'(b c))

#<syntax:main.rkt:38:5 (a b c)>

procedure

(stx-car stx) → syntax?

  stx : syntax?
Devuelve el objeto sintáctico situado a la izquierda de un stx-pair?.

Examples:
> (stx-car #'(a . b))

#<syntax:eval:20:0 a>

> (stx-car #'(hola adios casa))

#<syntax:eval:21:0 hola>

> (stx-car #'(bla))

#<syntax:eval:22:0 bla>

procedure

(stx-cdr stx) → syntax?

  stx : syntax?
Devuelve el objeto sintáctico situado a la derecha de un stx-pair?.

Examples:
> (stx-cdr #'(a . b))

#<syntax:eval:23:0 b>

> (stx-cdr #'(hola adios casa))

#<syntax:eval:24:0 (adios casa)>

> (stx-cdr #'(bla))

#<syntax:eval:25:0 ()>

value

stx-null : syntax? = #'()

Alias de #'().

Example:
> stx-null

#<syntax:main.rkt:48:19 ()>

procedure

(stx-list? stx) → syntax?

  stx : syntax?
Devuelve #t si el datum que contiene es list?.

Examples:
> (stx-list? #'(a . b))

#f

> (stx-list? #'(hola adios casa))

#t

> (stx-list? stx-null)

#t

procedure

(stx-list stx ...) → stx-list??

  stx : syntax?
Crea un objeto sintáctico que contiene una lista con cada uno de los stx ....

Examples:
> (stx-list #'a #'b #'c)

#<syntax (a b c)>

> (stx-list)

#<syntax ()>

> (stx-list #'(1 2 3) #'(a b c))

#<syntax ((1 2 3) (a b c))>

procedure

(stx-list* h-stx stx ...) → stx-list??

  h-stx : syntax?
  stx : syntax?
Crea un objeto sintáctico que contiene una lista no adecuada con h-stx y cada uno de los stx ....

Examples:
> (stx-list* #'a #'b #'c)

#<syntax:main.rkt:38:5 (a b . c)>

> (stx-list* #'a #'b #'(c d e f))

#<syntax:main.rkt:38:5 (a b c d e f)>

> (stx-list* #'(1 2 3) #'(a b c))

#<syntax:main.rkt:38:5 ((1 2 3) a b c)>

3 Operaciones sobre listas

procedure

(stx-length stx) → exact-nonnegative-integer?

  stx : stx-list?
Retorna el número de elementos de un stx-list?.

Examples:
> (stx-length #'())

0

> (stx-length #'(a b c d e))

5

> (stx-length #'(hola casa))

2

procedure

(stx-append stx1 stx2) → stx-list?

  stx1 : stx-list?
  stx2 : stx-list?
Concatena varias stx-list?.

Examples:
> (stx-append #'(a) #'(b c))

#<syntax:main.rkt:38:5 (a b c)>

> (stx-append #'(a b c d e) #'())

#<syntax:main.rkt:38:5 (a b c d e)>

> (stx-append #'(Para) #'(todo) #'(epsilon mayor) #'(que cero))

#<syntax:main.rkt:38:5 (Para todo epsilon mayor que cero)>

procedure

(stx-reverse stx) → stx-list?

  stx : stx-list?
Invierte el orden de los elementos de una stx-list?.

Examples:
> (stx-reverse #'(a b c))

#<syntax:main.rkt:38:5 (c b a)>

> (stx-reverse #'(hola y adios))

#<syntax:main.rkt:38:5 (adios y hola)>

> (stx-reverse #'(1 2 3 2 1))

#<syntax:main.rkt:38:5 (1 2 3 2 1)>

procedure

(stx-partition pred stx) → 
stx-list? stx-list?
  pred : procedure?
  stx : stx-list?
Parte stx en dos. Primero se retornan los elementos que devuelven #t usando pred, y luego se devuelve el resto.

Examples:
> (stx-partition stx-keyword? #'(#:hola a b #:adios))

#<syntax:main.rkt:38:5 (#:hola #:adios)>

#<syntax:main.rkt:38:5 (a b)>

> (stx-partition stx-list? #'((hola adios) casa manzana (1 2 3)))

#<syntax:main.rkt:38:5 ((hola adios) (1 2 3))>

#<syntax:main.rkt:38:5 (casa manzana)>

> (stx-partition stx-literal? #'(1 2 (pfff blaaa) hola (brrr brrr) "holi"))

#<syntax:main.rkt:38:5 (1 2 hola "holi")>

#<syntax:main.rkt:38:5 ((pfff blaaa) (brrr brrr))>

procedure

(stx-group-adjacent-by key stx same) → 
stx-list? stx-list?
  key : procedure?
  stx : stx-list?
  same : procedure?
Agrupa los elementos adyacentes de la lista que sean equivalentes. Dos elementos a y b son equivalentes si se verifica (same? (key a) (key b)).

Examples:
> (require racket/bool)
> (stx-group-adjacent-by stx-keyword? #'(#:hola #:adios b c d #:casa) (lambda (s1 s2) (not (xor s1 s2))))

#<syntax:main.rkt:38:5 ((#:hola #:adios) (b c d) (#:casa))>
